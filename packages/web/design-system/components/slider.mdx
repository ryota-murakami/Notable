## Usage

```tsx
import { Slider } from '@notable/web/design-system/components'

// Basic usage
<Slider defaultValue={[50]} max={100} step={1} />

// Controlled
const [value, setValue] = useState([50])

<Slider
  value={value}
  onValueChange={setValue}
  max={100}
  step={1}
/>

// Range slider
<Slider defaultValue={[20, 80]} max={100} step={1} />
```

## Common Patterns

### With Label and Value

```tsx
function SliderWithLabel() {
  const [value, setValue] = useState([50])

  return (
    <div className='space-y-2'>
      <div className='flex justify-between'>
        <Label>Volume</Label>
        <span className='text-sm text-muted-foreground'>{value[0]}%</span>
      </div>
      <Slider value={value} onValueChange={setValue} max={100} step={1} />
    </div>
  )
}
```

### Range Selection

```tsx
function PriceRangeSlider() {
  const [range, setRange] = useState([0, 1000])

  return (
    <div className='space-y-4'>
      <Label>Price Range</Label>
      <Slider value={range} onValueChange={setRange} max={2000} step={50} />
      <div className='flex justify-between text-sm'>
        <span>${range[0]}</span>
        <span>${range[1]}</span>
      </div>
    </div>
  )
}
```

### Custom Styling

```tsx
<Slider defaultValue={[50]} max={100} step={1} className='[&_[role=slider]]:h-4 [&_[role=slider]]:w-4' />
```

## Best Practices

1. **Clear labels**: Always provide context for what the slider controls
2. **Show values**: Display the current value(s) near the slider
3. **Appropriate steps**: Choose step values that make sense for the data
4. **Keyboard support**: Ensure arrow keys work properly
5. **Touch friendly**: Make touch targets large enough on mobile

## Accessibility

- Arrow keys for fine control
- Page Up/Down for larger jumps
- Home/End keys for min/max values
- ARIA attributes for screen readers
- Visual focus indicators
- Proper labeling
