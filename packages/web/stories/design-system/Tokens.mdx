Design tokens are the core visual properties that define the Notable Design System. They ensure consistency across all
components and make theming possible.

## Motion

Consistent motion creates a cohesive experience.

### Duration

- **Fast**: 150ms - Micro interactions
- **Normal**: 250ms - Standard transitions
- **Slow**: 350ms - Complex animations

### Easing

- **Ease In Out**: Default easing for most animations
- **Ease Out**: For elements entering the screen
- **Ease In**: For elements leaving the screen
- **Spring**: For playful, bouncy animations

## Usage in Code

```tsx
import { colors, spacing, typography, effects, motion } from '@notable/web/design-system/tokens'

// Using color tokens
const primaryColor = colors.brand[500]

// Using spacing tokens
const margin = spacing[4] // 16px

// Using typography tokens
const fontSize = typography.fontSize.body

// Using effect tokens
const shadow = effects.shadow.md

// Using motion tokens
const transition = `all ${motion.duration.normal} ${motion.easing.easeInOut}`
```
